# Varun Rana - Portfolio Website

🚀 **Modern, responsive portfolio showcasing Data Science, AI/ML, and Algorithmic Trading expertise.**

## 🎯 Live Demo

[View Portfolio](https://varun-portfolio-eta.vercel.app) *(After deployment)*

## ✨ Features

- **Stunning Dark Theme** with glassmorphism effects
- **Fully Responsive** - Mobile-first design
- **Interactive Animations** - Smooth scroll, hover effects
- **Project Filtering** - Filter by AI/ML, CV, or Trading
- **Skill Progress Bars** - Animated on scroll
- **Resume Download** - PDF download functionality
- **Contact Form** - Direct email integration

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts (Inter, JetBrains Mono)
- No frameworks - Fast & lightweight

## 🚀 Deployment

### Deploy to Vercel

1. Push to GitHub
2. Connect to Vercel
3. Auto-deploy!

```bash
git init
git add .
git commit -m "Initial commit: Portfolio website"
git remote add origin https://github.com/varunrana04/portfolio.git
git branch -M main
git push -u origin main
```
